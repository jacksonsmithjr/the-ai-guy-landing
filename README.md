# The AI Guy — Landing Page

Single-page landing page built with **HTML + CSS + vanilla JavaScript** (all inline in `index.html`).

## Edit these placeholders

- GHL booking calendar link: search for `BOOKING_URL`
- AI chat widget embed: search for `PASTE AI CHAT WIDGET EMBED CODE HERE`
- Lead form action/webhook: search for `REPLACE ACTION WITH YOUR WEBHOOK OR GHL FORM EMBED`

## Run locally

```bash
python -m http.server 5500
```

Then open `http://127.0.0.1:5500/`.
